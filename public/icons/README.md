# Icons Directory

Place your PNG icon files here.

## Recommended Structure

```
public/icons/
  ├── folder.png
  ├── documents.png
  ├── projects.png
  ├── photos.png
  ├── notes.png
  ├── code.png
  ├── design.png
  └── gaming.png
```

## Icon Specifications

- **Format**: PNG with transparency
- **Size**: 64x64px, 128x128px, or 256x256px
- **Color**: Full color or monochrome
- **Style**: Consistent across all icons

## Free Icon Resources

1. **Flaticon** - https://www.flaticon.com/
2. **Icons8** - https://icons8.com/
3. **Iconscout** - https://iconscout.com/
4. **Freepik** - https://www.freepik.com/
5. **Noun Project** - https://thenounproject.com/

## Quick Start

1. Download your preferred icons
2. Rename them appropriately (e.g., `folder.png`, `photos.png`)
3. Place them in this directory
4. Reference them in `src/App.tsx` using `/icons/filename.png`
